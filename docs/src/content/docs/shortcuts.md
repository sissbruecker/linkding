The following keyboard shortcuts are currently available:

| Action                                                                                    | Shortcut                            |
|-------------------------------------------------------------------------------------------|-------------------------------------|
| Add new bookmark                                                                          | <kbd>n</kbd>                        |
| Focus search input                                                                        | <kbd>s</kbd>                        |
| Navigate bookmarks                                                                        | <kbd>↑</kbd>, <kbd>↓</kbd>          |
| Toggle bookmark notes                                                                     | <kbd>e</kbd>                        |
